# SimpleSVG

Simple PHP class for creating SVG documents

## Instalation

```sh
php composer.phar require "fawno/simplesvg:@stable"
```

```php
<?php
  require 'vendor/autoload.php';

  use SimpleSVG\SimpleSVGDocument;
```
